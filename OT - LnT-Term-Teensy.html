Teensy 3.5 Microcontroller

As more automation gets integrated into ornamental turning, it is being delivered by microcontrollers like the Teensy from PJRC.

The Teensy grew from the work done by Italian professors to help students get better at developing control systems. They called this prototyping system the Arduino, and it has been greatly adopted by the maker community.

Fear not though : the programming efforts for this are being led by an awesome set of programmers who are posting their efforts on GitHub in the RoseEngine_SpindleAndAxis project.

 

Fun Fact: The name Arduino comes from a bar in Ivrea, Italy, where some of the founders of the project used to meet.
